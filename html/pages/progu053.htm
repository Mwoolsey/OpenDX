12.10 Component Manipulation

This section describes advanced routines for explicitly manipulating Field components. Since these routines expect the input to be a copy of the Object, they operate directly on the input without making another copy. For additional information on components, see Chapter 3. "Understanding the Data Model" in IBM Visualization Data Explorer User's Guide.

Object DXRename()

Renames a component in a Field. See DXRename.

Object DXSwap()

Exchanges two components in a Field. See DXSwap.

Object DXExtract()

Extracts a component from a Field. See DXExtract.

Object DXInsert()

Adds a component to a Field. See DXInsert.

Object DXReplace()

Adds a component from one Field to another. See DXReplace.

Object DXRemove()

Deletes components from a Field. See DXRemove.

Object DXExists()

Determines whether if a component exists in a Field. See DXExists.
