DXNewString

Function

Creates a new String Object.

Syntax

#include <dx/dx.h>


String DXNewString(char *s)

Functional Details

The new String Object is initialized with a copy of the specified NULL-terminated string s.

The Object created can be deleted with DXDelete. See 4.2 , "Memory Management".

Return Value

Returns the Object or returns NULL and sets an error code.

See Also

DXGetString

12.4 , "String Class".
