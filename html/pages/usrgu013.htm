Backward Incompatibilites

With significant updates to Java Explorer, most applets and html files will need to be re-done. If you are using Java Explorer, you will probably want to regenerate the html and java file then re-install it.

Since Categorize has a new option for not sorting the lookup table, if you plan on moving networks to later versions, do not set this option.

Using the new test for Inquire in an older version will produce an error within DX.

Creating a multi-line annotation with Text in an older version will just produce an annotation with the \n shown in a single line annotation.

Adding the tag to resize an image with ImageMagick in WriteImage will produce an error in previous versions of DX.
